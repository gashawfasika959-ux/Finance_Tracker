# Finance Tracker Web App 💰

A **simple yet powerful web application** to track personal income and expenses. Users can add transactions, view summaries, and monitor their balance in real time. Data is saved in the browser using **LocalStorage**, so it persists across sessions.

---

## 🚀 Features

- Add income and expense transactions  
- View total income, total expense, and balance  
- List of all transactions  
- Persistent data using LocalStorage  
- Simple, responsive, and user-friendly design  
- Optional: Dark mode toggle, categories, charts  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Data Storage:** LocalStorage  
- **Optional Upgrade:** Chart.js for graphs
